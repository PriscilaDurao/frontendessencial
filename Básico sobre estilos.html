Básico sobre estilos.
Aqui você vai encontrar uma breve introdução sobre folhas de estilo em CSS. CSS é abreviação para o termo em inglês Cascading Style Sheets, o que pode ser traduzido para português como Folha de Estilo em Cascata. Essa linguagem de desenvolvimento web muito utilizada para fazer a apresentação visual, q qual chamamos de estilos, de documentos escritos em uma linguagem de marcação de texto (HTML).
